Dans ce projet nous avons réalisé le jeu de Nim et TicTacToe (TicTacToe Normal et TicTacToe avec Aide).
Le but du jeu est de pouvoir choisir un des 3 jeux suivants et de de choisir aussi un adversaire qui est soit:
- un autre humain 
- Un robot aléatoire
- Un robot doué d'intelligence

Nous avons 4 différents packages :games, players, plays et main
Dans le package games vous aurez les classes et interfaces suivantes:
- AbstractGame.java
- Game.java
- Nim.java
- TicTacToe.java
- TicTacToeWithHints.java

Dans le package players vous trouverez les classes et interfaces suivantes:
- Human.java
- NegamaxPlayer.java
- RandomPlayer.java
- Player.java

Dans le package plays vous trouverez les classe suivante:
-Orchestrator.java

Dans le package main on aura :
- Notre classe Main.java qui est la classe exécutable c'est-à-dire pemettant de jouer un jeu
- Et la classe Test.java qui permet de faire les test
